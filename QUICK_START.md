## üìú Enhanced History

### New History Features

1. **Search Your History**
   - Click the history icon to open the panel
   - Use the search bar to filter prompts by content
   - Results update in real-time

2. **Duplicate Prompts**
   - Click the copy icon next to any history item
   - Instantly loads it for editing
   - Creates a new entry with current timestamp

3. **Delete with Confirmation**
   - Click the trash icon on any item
   - Confirm before deletion (safety first!)
   - Or clear all history at once

4. **Import & Export**
   - **Export:** Download your history as JSON or CSV
   - **Import:** Upload a previously exported JSON file
   - Merge with existing or replace entirely

### How to Export/Import

**Export:**
1. Open history panel
2. Click the download icon
3. Choose from the dropdown (JSON/CSV)
4. File downloads automatically

**Import:**
1. Open history panel
2. Click the upload icon
3. Select your JSON file
4. Prompts are merged with existing history

## üêõ Troubleshooting

### Common Issues:

**Input not accepted?**
- Check character count (min 10, max 5000)
- Ensure text isn't just spaces
- Look for red validation messages

**History not showing?**
- Make sure you've generated at least one prompt
- Check if browser storage is enabled
- Try refreshing the page

**Import not working?**
- Verify file is JSON format
- Check file size (max 10MB)
- Ensure file isn't corrupted

**Keyboard shortcuts not working?**
- Make sure no input field is focused
- Check if browser extensions are interfering
- Try pressing `?` to see available shortcuts
